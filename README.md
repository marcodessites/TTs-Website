# TTs-Website

This repository contains two projects:

- `TTs Site/` — React/Vite children’s book storefront
- `Mars Extension/` — Chrome extension with an Auto Correct toggle

## Vercel Deployment

The `TTs Site` folder is configured for deployment with Vercel using `vercel.json`.

## Structure

- `TTs Site/` contains the web app and Vite configuration
- `Mars Extension/` contains the extension manifest, popup, and content script
